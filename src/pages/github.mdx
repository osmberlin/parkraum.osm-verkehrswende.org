# Github

Der Quellcode und die Daten der hier beschriebenen Projekte sind an unterschiedlichen Stellen zu finden…

## Diese Website

- AstroJS mit React- und JavaScript Komponenten unter [github.com/osmberlin/parkraum.osm-verkehrswende.org](https://github.com/osmberlin/parkraum.osm-verkehrswende.org)

## Projekt Parkraum Berlin (Vector Tiles)

- Prozessierung mit osm2gpsql und PostGIS, Docker Container unter [github.com/osmberlin/osm-parking-processing](https://github.com/osmberlin/osm-parking-processing)
- Export der Daten auf den Seiten der jew. Region
- Kartendarstellung der Daten im Radverkehrsatlas

## Projekt Prototyp Neukölln (Parkraumkarte)

- Prozessierung mit Python und QGIS unter [github.com/SupaplexOSM/street_parking.py](https://github.com/SupaplexOSM/street_parking.py)
- Export der Daten [auf der Prototyp Neukölln-Seite](/project-prototype-neukoelln/data)
- Kartendarstellung der Daten [auf dieser Website](https://github.com/osmberlin/parkraum.osm-verkehrswende.org/tree/main/src/pages/project-prototype-neukoelln) über Maplibre GL JS

## Radverkehrsatlas

- Kartendarstellung für der Vector Tiles mit Additionstool unter [github.com/FixMyBerlin/atlas-app](https://github.com/FixMyBerlin/atlas-app)
